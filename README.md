# Programa Poliperros
Aplicación de escritorio desarrollada en Python para la gestión de pedidos y entregas.

## Requisitos
- Sistema operativo Windows

## Instalación
1. Descargar el archivo `poliperros.exe`
2. Ejecutar el archivo descargado
3. Seguir las instrucciones en pantalla

> No es necesario tener Python instalado.

## Uso
El programa se ejecuta desde una ventana de consola y permite:
- Registrar pedidos
- Mostrar pedidos
- Eliminar pedidos

## Notas
Si Windows muestra un mensaje de seguridad, seleccionar  
**"Más información" → "Ejecutar de todas formas"**.
